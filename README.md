# Sharkord Lava Plugin

A plugin to bring Discord-like music bot to [Sharkord](https://github.com/Sharkord/sharkord) powered by [Lavalink fork](https://github.com/Salaron/Lavalink) with RTP support.

> [!WARNING]
> This plugin is not compatible with Lavalink instances without RTP support!
